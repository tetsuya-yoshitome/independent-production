# データベースで商品管理①

## 概要
このシステムでは、販売した商品の管理ができます。
商品の新規登録、編集、削除ができます。
また登録した商品の検索も可能です。
検索結果画面から同日に売れた商品一覧を、売上件数と売上合計とともに表示させることもできます。

## 主な機能
- ログイン・ログアウト機能

- 商品一覧画面

- 商品新規登録、編集、削除

- 商品販売管理機能

- 商品検索機能

- 同日販売一覧画面

## 開発環境

PHP 8.1.6

MySQL 15.1

Laravel 8.83.3
